## Run Locally

Clone the project

```bash
  git clone https://github.com/yuelinwen/safe-office-reservation.git
```

Go to the project directory

```bash
  cd fragments
```

Install dependencies

```bash
  npm install
```

Test and check errors with lint

```bash
  npm run lint
```

Start the server (choose one of command to start)

```bash
  npm start
  npm run dev
  npm run debug
```
